## 1. 创意名称 + 创意介绍

**创意名称：FlashInterview —— 碎片时间刷面经的 PWA 抽认卡**

### 想解决什么问题

求职者在备战面试时，会积累大量 markdown 格式的面经笔记，但这些文件都躺在电脑本地硬盘里。手机端的碎片时间（地铁、排队、睡前）想复习却打不开，而且就算能打开，逐篇阅读效率极低——看完「感觉自己会了」，但其实只是被动浏览，没有主动回忆，记忆留存率很低。

### 为什么会想到做这个

我自己是求职者，深有体会——每次面试前想复习面经，对着电脑坐不住，地铁上却什么都看不了。**Anki 之类的抽认卡工具**其实早已验证了"主动回忆"对记忆的巨大提升，但门槛太高：要手动一张张录入卡片、设置间隔重复参数。我只是想把已经写好的面经笔记**自动转成抽认卡**，把"录入"的成本降到零。

### 大概是什么产品

一个 PWA（渐进式 Web 应用）形态的面试抽认卡工具。Python 脚本一次性解析 9 份 markdown 面经 → 生成结构化 JSON → PWA 以抽认卡模式呈现，支持翻转、标记熟度、智能间隔重复筛选、按公司精准定位。**零后端、纯离线，装到手机主屏幕就是 App**。

---

## 2. 目标用户及痛点

### 面向哪些用户

- **正在求职/找实习的开发者**：尤其是技术岗（Java 后端、AI Agent、前端、算法）
- **面试密集期的求职者**：短时间内要面多家公司不同岗位
- **已经在整理面经笔记的人**：Markdown 笔记积累了一堆，但从未真正消化

### 在什么场景下使用

- 🚇 **通勤路上**：地铁、公交，单程 30-60 分钟
- 🕐 **排队等待**：等餐、等咖啡、等电梯
- 🛏️ **睡前的 5-10 分钟**：躺着翻几道题
- ☕ **摸鱼间隙**：会议间隙、外卖等待

### 当前痛点

1. **手机看不了本地 markdown**：面经都在电脑硬盘里，手机端无解
2. **逐篇阅读效率低**：从第一篇读到最后只是被动浏览，没有主动回忆机制
3. **不知道哪家公司高频考什么**：9 份面经混在一起看，缺少按公司/类型的筛选
4. **复习进度不沉淀**：每次重新读一遍，但已掌握的和没掌握的分不清，时间浪费在会的题上
5. **隐私顾虑**：面试笔记涉及大量项目细节和个人思考，上传云端不放心

---

## 3. 价值与意义

### 效率提升（核心价值）

- **激活 80% 闲置的碎片时间**：把原本"看不了"的场景变成"高效刷题"场景
- **间隔重复算法**：标记"熟了"的题减少出现，"不熟"的加权 ×3，**把时间集中在薄弱点**
- **3D 翻面强制主动回忆**：先自己想答案，再翻面对比，比直接阅读记忆留存率高 3-5 倍（认知科学验证）
- **按公司精准定位**：明天面美团，今晚只刷美团的题，针对性极强

### 隐私价值（差异化亮点）

- **数据 100% 在本地**：所有面经数据存于 localStorage，零服务器、零上传
- **适合敏感场景**：面试笔记包含个人项目经历、公司评价、内部信息等敏感内容
- **PWA 离线可用**：地铁无网、电梯无信号，照样能刷

---

## 4. 产品形态与差异化亮点

### 核心特性

1. 🎴 **3D 翻转抽认卡**：CSS 3D transform 模拟真实抽认卡手感
2. 🧠 **智能间隔重复**：review ×3、unknown ×2、known ×1 权重抽题
3. 🏢 **按公司/类型筛选**：美团/蚂蚁/腾讯/阿里/宇树/成都智信，技术/行为/系统设计
4. 💾 **进度永不丢失**：localStorage 持久化，关掉再开熟度标记全保留
5. ➕ **在线添加题目**：面试现场想到新题？直接在应用里添加，自动入库
6. 📱 **PWA 主屏幕安装**：添加到主屏幕，全屏运行，离线可用，像原生 App

### 技术栈

- **前端**：纯 HTML + CSS + vanilla JS（无框架依赖，零运行时开销）
- **预处理**：Python 3（解析 9 份 markdown 面经为结构化 JSON）
- **离线**：Service Worker 缓存所有资源
- **持久化**：localStorage

### 与现有方案的对比

| 方案 | 痛点 |
|------|------|
| 微信收藏笔记 | 没有抽认卡模式，被动阅读 |
| Anki | 录入成本极高，不支持 markdown 批量导入 |
| Notion 移动端 | 依赖网络、隐私顾虑、同样无主动回忆 |
| 单纯刷题 App | 题目固定，无法用自己的面经 |

**FlashInterview 的差异化**：**零成本导入（自动解析 markdown） + 隐私本地化 + 抽认卡主动回忆**。
